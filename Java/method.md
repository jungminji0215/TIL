# static 메서드와 인스턴스 메서드

- 두 메서드의 차이점
- static 메서드는 언제 사용하는 것이 좋을까?

<br>

## ✅ static 메서드 (클래스 메서드)

객체를 생성하지 않고 메서드를 사용할 수 있다.

static 메서드에서 인스턴스 변수와 인스턴스 메서드 둘 다 사용할 수 없다.

<br>

## ✅ 인스턴스 메서드

객체를 생성한 다음에 메서드를 사용할 수 있다. 

<br>

## ✅ static 메서드는 언제 사용하는 것이 좋을까?

인스턴스 멤버를 사용하지 않는 메서드에 static을 붙이는것이 좋다.
