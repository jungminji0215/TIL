# 💎 모의고사

[코딩테스트 연습 - 모의고사](https://programmers.co.kr/learn/courses/30/lessons/42840)

## 🔎 파이썬

### 📌 정답 코드

```python
def solution(answers):
    score = []
    winner = []
    
    student = [[1,2,3,4,5] ,
               [2, 1, 2, 3, 2, 4, 2, 5] ,
               [3, 3, 1, 1, 2, 2, 4, 4, 5, 5]]
    
    for i in range(3):
        count = 0

        for j in range(len(answers)):
            if answers[j] == (student[i][j % len(student[i])]):
                count += 1 
                
        score.append(count)

    for i in range(3):
        if score[i] == max(score):
            winner.append(i+1)
            
    return winner
```

### 💡 유형

완전 탐색

### 💡 풀이

**student[i][j % len(student[i])]** 식을 도출해 내는 것에서 시간이 오래 걸렸다. 배열이 순환할 때(?) 어떤 식으로 코드를 짜야 하는지 기억해두자.
